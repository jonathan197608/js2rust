## 关键发现

### 主要不准确项（需立即修正）

1. **`instanceof` 运算符**: 文档标记 ✅ 已实现，实际生成 `@compileError` (expr.rs:511)
2. **`void` 运算符**: 文档标记 ✅ 已实现，实际生成错误信息 (expr.rs:1814)
3. **`delete` 运算符**: 文档标记 ✅ 已实现，实际生成错误信息 (expr.rs:1814)
4. **`obj[key]` 动态属性访问**: 文档标记 ✅ 已实现，实际仅支持数字字面量索引 (expr.rs:151-167)
5. **`Date.UTC()`**: 文档标记 ✅ 已实现，实际生成 `@compileError` (expr.rs:1614-1615)

### 主要遗漏项（需添加到文档）

1. **生成器函数 (`function*`, `yield`, `yield*`)**: 完全未提及
2. **异步生成器 (`async function*`)**: 完全未提及
3. **逻辑赋值运算符 (`&&=`, `||=`, `??=`)**: ES2021 特性，未提及
4. **指数赋值运算符 (`**=`)**: ES2016 特性，未提及
5. **`arguments` 对象**: 传统函数参数对象，未提及
6. **`import.meta`**: ES 模块元数据，未提及
7. **`Symbol`**: ES6 符号，未提及
8. **`WeakMap` / `WeakSet`**: 弱引用集合，未提及
9. **`Intl.*`**: 国际化 API，完全未提及

### 实现状态不确定项（需人工核对）

1. **`typeof` 运算符**: 生成 `@typeName(@TypeOf(x))`，行为与 JS `typeof` 不同
2. **`null` 字面量**: 代码中 `NullLiteral` 返回 `None` 类型，可能导致类型推断错误
3. **可选链 `?.`**: 生成 `if (obj) |v| v.prop else null`，可能存在空指针解引用风险
4. **闭包可变捕获**: 生成 `self.x.*` 解引用，可能导致 Zig 借用检查器错误
5. **模板字符串插值**: 使用 `allocPrint` 分配内存，可能存在内存泄漏
